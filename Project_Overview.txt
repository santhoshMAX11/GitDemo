Project Name: Bookstore Web App Manual Testing

Objective:
To test an online bookstore web application manually using industry-standard tools and Agile methodology.

Modules to Test:
- User Login & Signup
- Book Search & Filters
- Cart & Checkout
- Order History

Tools Used:
- JIRA (for task tracking and bug reporting)
- Qase.io (for writing and executing test cases)
- GitHub (to upload all project files)
- Git Bash: For Git operations (init, add, commit, push)
- Agile (for planning and execution)

Methodology:
We will follow Agile SDLC and STLC process for this project.
